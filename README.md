## 1. サービス概要

- Make Mazes!（略してメイメイ🐏）は誰でも簡単に迷路を作成し、印刷して遊ぶことができるアプリです。

- 迷路を「作る」「解く」「共有」することで家族や友人や、迷路が好きな誰かと、楽しくリラックスできるキッカケをつくることが目的です。

### 補足：迷路のつくり方　（ページ: 「迷路をつくる！」迷路エディタ操作） 
1. 🚪 まず指定された迷路枠上のセルに「入口」と「出口」を決めます。
2. ✏️「描画」を選択し、入口のセルから１本線で好きな絵（正解ルート）を描いて出口までつなぎます。
3. ♻️ 入口と出口がつながった後で、「迷路を生成する」を押すだけ。正解ではない通路が自動生成されて迷路となります。
4. 🖨️ プレビューページで確認後、気に入った迷路になっていれば、印刷したり画像ダウンロードして利用することができます。
5. 📝 印刷用紙に「迷路のタイトル」、「作ったあなたからのひとこと」、「解いた人用の感想欄」も任意で加えることもできます。

※ 詳細は「つかい方ガイド」に記載されています。

## 2. このサービスへの思い・作りたい理由
みなさんも感じていることと思いますが<br>
日々の生活のなかで __「大切な家族との時間、友人との時間」__ があると思います。<br>
できればその<b>大切な時間を、楽しくリラックスして共有できるサービス</b>を作りたいと思い、このアプリを作りました。

### < 作るきっかけとなったのはズバリ >
__「子どもたちと一緒に迷路パズルを解いたときの楽しさ」__ ☺️<br>
一緒に遊んだときに思いました。<br>
<b>「迷路って年齢とか経験とか関係なく楽しめていいなぁ」</b>と。

今は世界中に色々なゲームやサービスはあるけれど、<br>
シンプルな迷路は、年齢や経験、生き方、性別や国籍も関係なく楽しめるんじゃないかと。

大人も子供も熱中したり、ほっこりできる迷路を、<br>
誰でも作れて、みんなでシェアして遊べたらいいなと思って作りました。

## 3. ユーザー層について
- 迷路が好きなひと
- 子供と一緒に迷路を作って遊びたい親
- 紙で配布できるオリジナル迷路を作りたいひと
- 子供の教育に携わるひと

## 4. サービスの利用イメージ
1. 迷路を作りたいユーザーが「迷路」を作成する
2. 作成した迷路をシェアして他のユーザーに届ける
3. 他のユーザーが迷路を解いて楽しみ、作成したユーザーと迷路の感想や意見交換をおこなう
4. 迷路を解いたユーザーも「迷路を作る」ユーザーとなり、同様に①②③をおこなう♻️
   
__→ このアプリを通して、ユーザー同士が親しくなり、<br>一緒にいてリラックスできるキッカケを作る☺️☺️__ 

## 5. サービスの差別化ポイント
この迷路アプリのポイントは、<br>
一般的な迷路の自動生成アプリと異なり
__「ユーザーが作成した迷路のゴールまでの道筋をもとに迷路が作成される」__ ことです。<br>
__ゴールの道筋を意図的にイラスト状にすると、「ゴール図＝制作者が描いたイラスト」__ となります。<br>

イラストになる<b>趣向を凝らした迷路を作成</b>できたり、<br>
それを解いて<b>完成図を見る楽しみ</b>があり、<br>
<b>幅広いユーザー層が興味を持って利用できる</b>と考えています。

## 6. 主要機能について
### MVPリリース版での機能
迷路エディタ上での手描き迷路作成
- 迷路枠上に入口・出口を指定し、枠内に入口から出口までの経路を描画し、迷路生成する機能
- 迷路描画時の線の取り消し、やり直し機能（Undo / Redo）
- 迷路の問題と解答の画像の生成機能
- 生成した迷路の短期保持機能
- 生成迷路を印刷や画像ダウンロードして利用できる機能

### 本リリース版での機能（案）
ユーザー登録による以下の機能の追加（登録なし = ゲスト：MVPの機能を提供）
- ログインユーザー：
- 迷路データのCRUD機能（追加・編集・削除・一覧表示）
- 作成する迷路の難易度選択、迷路枠形状の追加、枠線の配色変更等の機能
- ユーザー設定のCRUD機能（アカウント変更/削除）
- X、インスタグラムへの投稿機能
- LINE招待機能

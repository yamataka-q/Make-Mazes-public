# ♻️ Make Mazes!
__ワクワクできる迷路を作ろう！解こう！みんなでシェアしよう！__ 

## サービス概要
♻️ Make Mazes!（略してメイメイ🐏）は、誰でも簡単に迷路を作成、印刷して遊ぶことができるアプリです。<br>
使い方は簡単で、入口と出口を決めてゴールまで一本線で好きな絵（道）を描いてつなぎ、作成ボタンを押すだけ。迷路が作成できます。<br>
迷路を「作る」「解く」「共有」することで、家族や友人や、さらに迷路が好きな誰かと、楽しくリラックスできるキッカケをつくりましょう🖼️🖊️

## このサービスへの思い・作りたい理由
みなさんも感じていることと思いますが、日々の生活のなかに __「大切な家族との時間、友人との時間」__ がありますよね。できればその大切な時間を、楽しくリラックスして共有できるためのサービスを作りたいと思っていました。<br>
このサービスを作るきっかけとなったのはズバリ __「子どもたちと一緒に迷路パズルを解いたときの楽しさ」__ ☺️<br>
そのときに思いました。「迷路って年齢とか経験とか関係なく楽しめていいなぁ」と。<br>
今は世界中に色々なゲームやサービスはあるけれど、シンプルな迷路は、年齢や経験、生き方、性別や国籍も関係なく楽しめるのではないかと。大人も子供も熱中したり、ほっこりできる迷路を、誰でも作れてみんなでシェアして遊べたらいいなと思い、作ろうと思いました。

## ユーザー層について
- 迷路が好きなひと
- 子供と一緒に迷路を作って遊びたい親
- 紙で配布できるオリジナル迷路を作りたいひと
- 子供の教育に携わるひと

## サービスの利用イメージ
1. 迷路を作りたいユーザーが「迷路」を作成する
2. 作成した迷路をシェアして他のユーザーに届ける
3. 他のユーザーが迷路を解いて楽しみ、作成したユーザーと迷路の感想や意見交換をおこなう
4. 迷路を解いたユーザーも「迷路を作る」ユーザーとなり、同様に①②③をおこなう♻️
   
__→ このアプリを通して、ユーザー同士が親しくなり、一緒にいてリラックスできる関係を築くキッカケを作る！☺️☺️__ 

## ユーザーの獲得について
本リリースでSNSの投稿機能でシェアされ、興味を持ったユーザーを自然に獲得します。

## サービスの差別化ポイント・推しポイント
このサービスのポイントは、__「ユーザーが作成した迷路のゴールまでの道筋をもとに迷路が作成される」__ ことです。<br>
__ゴールの道筋を意図的にイラスト状にすると、「ゴール図＝制作者が描いたイラスト」__ となります。<br>
イラストになる趣向を凝らした迷路を作成できたり、それを解いて完成図を見る楽しみがあり、幅広いユーザー層が興味を持って利用できると思います。

## 機能候補
### MVPリリース機能
Canvas上での手描き迷路作成
- 迷路枠上に入口・出口を指定し、枠内に手入口から出口までの経路を描画し、迷路生成する機能
- 迷路描画時の線の取り消し、やり直し機能（Undo / Redo）
- サーバでの印刷方式（短期保持）
- 迷路の問題と解答の画像を生成（その後、ユーザーが「ダウンロード」して 印刷したり、SNSに投稿）

 ### 本リリース機能
ユーザ登録による以下の機能の追加（登録なし = ゲスト：MVPの機能を提供）
- ログインユーザー：
- 迷路データのCRUD機能（追加・編集・削除・一覧表示）
- 作成する迷路の難易度、迷路枠形状の追加、枠線の配色変更
- ユーザー設定のCRUD機能（アカウント変更/削除）
- X、インスタグラムへの投稿機能
- LINE招待機能

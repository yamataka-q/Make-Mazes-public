# ♻️ Make Mazes!

## 本番環境
https://make-mazes.onrender.com

## サービス概要
__ワクワクできる迷路を作ろう！解こう！みんなでシェアしよう！__ 

- ♻️ Make Mazes!（略してメイメイ🐏）は、誰でも簡単に迷路を作成、印刷して遊ぶことができるアプリです。

- 🧩使い方は簡単で、迷路枠内に入口かと出口を決めて、入口から１本線で好きな絵（正解ルート）を描いて出口までつなぎ、作成ボタンを押すだけ。迷路が作成できます。<br>

- 🖨️プレビューページで確認し、気に入った迷路になっていれば、印刷したり画像ダウンロードしてして利用することができます。🖨️

- 📝迷路には迷路のタイトルや、作ったあなたからのひとこと、解いた相手への感想欄も任意で添えることもできます。

- 👥迷路を「作る」「解く」「共有」することで、家族や友人や、さらに迷路が好きな誰かと、楽しくリラックスできるキッカケをつくりましょう！

## このサービスへの思い・作りたい理由
みなさんも感じていることと思いますが、<br>
日々の生活のなかに __「大切な家族との時間、友人との時間」__ がありますよね。<br>
できればその<b>大切な時間を、楽しくリラックスして共有できるためのサービスを作りたい</b>と思っていました。

このサービスを作るきっかけとなったのはズバリ <br>
__「子どもたちと一緒に迷路パズルを解いたときの楽しさ」__ ☺️<br>
そのときに思いました。<br><b>「迷路って年齢とか経験とか関係なく楽しめていいなぁ」</b>と。

今は世界中に色々なゲームやサービスはあるけれど、<br>シンプルな迷路は、年齢や経験、生き方、性別や国籍も関係なく楽しめるのではないかと。

大人も子供も熱中したり、ほっこりできる迷路を、<br>誰でも作れてみんなでシェアして遊べたらいいなと思って作ろうと思いました。

## ユーザー層について
- 迷路が好きなひと
- 子供と一緒に迷路を作って遊びたい親
- 紙で配布できるオリジナル迷路を作りたいひと
- 子供の教育に携わるひと

## サービスの利用イメージ
1. 迷路を作りたいユーザーが「迷路」を作成する
2. 作成した迷路をシェアして他のユーザーに届ける
3. 他のユーザーが迷路を解いて楽しみ、作成したユーザーと迷路の感想や意見交換をおこなう
4. 迷路を解いたユーザーも「迷路を作る」ユーザーとなり、同様に①②③をおこなう♻️
   
__→ このアプリを通して、ユーザー同士が親しくなり、<br>一緒にいてリラックスできるキッカケを作る☺️☺️__ 

## ユーザーの獲得について
本リリースでSNSの投稿機能でシェアされ、興味を持ったユーザーを自然に獲得します。

## サービスの差別化ポイント・推しポイント
このサービスのポイントは、<br>
__「ユーザーが作成した迷路のゴールまでの道筋をもとに迷路が作成される」__ ことです。<br>
__ゴールの道筋を意図的にイラスト状にすると、「ゴール図＝制作者が描いたイラスト」__ となります。<br>

イラストになる<b>趣向を凝らした迷路を作成</b>できたり、<br>
それを解いて<b>完成図を見る楽しみ</b>があり、<br>
<b>幅広いユーザー層が興味を持って利用できる</b>と考えています。

## 機能候補
### MVPリリース機能
Canvas上での手描き迷路作成
- 迷路枠上に入口・出口を指定し、枠内に手入口から出口までの経路を描画し、迷路生成する機能
- 迷路描画時の線の取り消し、やり直し機能（Undo / Redo）
- サーバでの印刷方式（短期保持）
- 迷路の問題と解答の画像を生成（その後、ユーザーが「ダウンロード」して 印刷したり、SNSに投稿）

 ### 本リリース機能
ユーザ登録による以下の機能の追加（登録なし = ゲスト：MVPの機能を提供）
- ログインユーザー：
- 迷路データのCRUD機能（追加・編集・削除・一覧表示）
- 作成する迷路の難易度、迷路枠形状の追加、枠線の配色変更
- ユーザー設定のCRUD機能（アカウント変更/削除）
- X、インスタグラムへの投稿機能
- LINE招待機能

# Feature Summary

## 概要
Make Mazes! は、入口から出口までの正解ルートをユーザーが描き、
そのルートをもとにオリジナル迷路を生成できるアプリです。

作成した迷路は、問題・解答・印刷レイアウトとして確認でき、
PNG画像のダウンロードや印刷にも対応しています。

## コア機能

### 1. 入口と出口の指定
迷路の外周セルから入口と出口を指定できます。  
角セルは選択できません。

### 2. 正解ルートの描画
入口から出口まで、正解ルートを1本の連続経路として描画できます。  
ユーザーが描いたルートが、迷路生成の土台になります。

### 3. 迷路の自動生成
正解ルート完成後、「迷路を生成する」を実行すると、
正解ではない通路を含む迷路が自動生成されます。

## エディタ機能

### 操作モード
- 入口指定
- 出口指定
- 描画

### 描画編集
- 1セル戻す
- 描画単位で戻す
- やり直す
- 経路を消す
- 全部消す

### 補助機能
- 描画補助キー（十字キー）
- 状態表示
- 印刷用の追加情報設定

## プレビュー機能
生成した迷路は、以下の3種類で確認できます。

- 問題表示
- 解答表示
- 印刷レイアウト表示

## 出力機能
Make Mazes! では、生成した迷路を以下の形で利用できます。

- 問題用PNGダウンロード
- 解答用PNGダウンロード
- A4印刷

## 印刷用の追加情報
印刷時には、必要に応じて以下の情報を追加できます。

- 迷路タイトル
- 制作者からひとこと
- 解いた人の感想欄

## 一時保存
エディタ状態やプレビュー情報は、一時的に保存されます。  
MVP時点では、短時間（30分）の再確認やページ移動を想定した構成です。

## MVPで提供している内容
- 手描きで正解ルートを作成
- 迷路の自動生成
- 問題 / 解答 / 印刷のプレビュー
- PNGダウンロード
- 印刷対応
- 一時保存

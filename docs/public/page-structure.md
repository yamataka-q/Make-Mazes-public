# Page Structure

## 概要
Make Mazes! は、少ないページ数で
「迷路を作る」「確認する」「出力する」までを完結できる構成を目指しています。

## ページ一覧

### 1. TOPページ
アプリの概要と入口になるページです。  
迷路作成ページと、つかい方ガイドへの導線があります。

### 2. つかい方ガイド
初めて使う人向けに、基本の操作手順や各ボタンの役割を説明するページです。

### 3. 迷路エディタ
入口・出口の指定、正解ルートの描画、迷路生成を行う中心ページです。

### 4. プレビューページ
生成した迷路を確認するページです。  
以下の3種類の表示があります。

- 問題表示
- 解答表示
- 印刷レイアウト

## 各ページの役割

### TOPページの役割
- アプリの目的を伝える
- 迷路作成への導線を示す
- つかい方ガイドへの導線を示す

### つかい方ガイドの役割
- 初回利用時の迷いを減らす
- 入口 / 出口 / 描画 / 取り消し / 生成 / 出力までの流れを説明する

### 迷路エディタの役割
- 迷路作成の中心
- 正解ルートを入力する場所
- 迷路生成の起点
- 印刷用追加情報の設定場所

### プレビューページの役割
- 生成結果の確認
- 問題用 / 解答用の切り替え
- 印刷前の確認
- 画像ダウンロードや印刷への導線

## 基本の利用フロー

1. TOPページから迷路作成ページへ進む
2. 必要に応じて、つかい方ガイドを確認する
3. 迷路エディタで入口と出口を指定する
4. 正解ルートを描画する
5. 迷路を生成する
6. プレビューページで問題・解答・印刷レイアウトを確認する
7. 印刷またはPNG画像をダウンロードして利用する

## ページ構成上のポイント
Make Mazes! では、ページ数を増やしすぎず、
利用者が流れを追いやすい構成を意識しています。

- TOPで概要を理解できる
- ガイドで操作方法を確認できる
- エディタで作成に集中できる
- プレビューで利用形態ごとに確認できる

## 補足
今後の本リリースでは、保存機能や共有機能の追加に合わせて、
ページ構成や導線もさらに見直していく予定です。

# UI Notes

## 概要
Make Mazes! は、迷路を「作る」「確認する」「印刷する」までを、
できるだけ迷わず進められるUIを目指して設計したアプリです。

公開中の画面は、主に以下の4つで構成されています。

- TOPページ
- つかい方ガイド
- 迷路エディタ
- プレビューページ（問題 / 解答 / 印刷）

## UI全体の方針
このアプリでは、以下の点を大切にしています。

1. 初めて見た人でも、何ができるアプリかすぐ伝わること
2. 迷路を作る流れが自然につながること
3. 描画や確認の操作で迷いにくいこと
4. 印刷や画像出力まで一連の流れとして扱えること

## 各画面の役割

### TOPページ
TOPページでは、アプリの目的や楽しみ方が最初に伝わるようにしています。  
「迷路をつくる！」と「つかい方ガイド」の導線を用意し、
初回利用者でも次の行動を選びやすい構成にしています。

### つかい方ガイド
つかい方ガイドでは、入口・出口の指定、描画、取り消し、迷路生成、
プレビュー確認までの流れを段階的に説明しています。  
迷路エディタを触る前に確認できるため、初回の迷いを減らす役割があります。

### 迷路エディタ
迷路エディタはこのアプリの中心画面です。  
迷路のマス目と、操作パネル・迷路生成パネル・簡易ガイドを分けて配置し、
「どこで描くか」と「どこで操作するか」が明確になるようにしています。

### プレビューページ
プレビューページでは、生成された迷路を
「問題」「解答」「印刷レイアウト」の3種類で確認できます。  
表示を切り替えながら、印刷やPNGダウンロードまで進められる構成です。

## 主なUI上の工夫

### 外周セルのみ入口と出口を指定
迷路らしい見た目と操作のわかりやすさのため、
入口と出口は外周セルに限定しています。角セルは選べません。

### 描画の流れを段階化
入口・出口が決まるまでは描画できず、
描画は入口からのみ開始できます。  
これにより、迷路作成の流れを理解しやすくします。

### 描画補助キーの導入
スマホやタブレットなどで小さなセルを操作しづらい場面に対応するため、
描画補助キーとして十字キーUIを用意しました。

### 印刷情報の追加
迷路のタイトル、制作者コメント、解いた人の感想欄を
任意で印刷用レイアウトへ追加できるようにしています。  
遊ぶだけでなく、配布や共有もしやすく、ユーザー体験の向上を意識しています。

## モバイル対応で意識したこと
Make Mazes! では、モバイル操作も意識して調整を進めています。

- 小さいセルでも操作しやすいように補助UIを用意
- 画面幅に応じてセルサイズや余白を調整
- タップ操作時の誤操作を減らす方向で改善
- プレビューや印刷確認でも見やすい余白構成を意識

## 印刷・出力まわりの考え方
このアプリでは、迷路を「作って終わり」にせず、
実際に遊べる形まで持っていくことを大切にしています。

- 問題用と解答用を分けて確認できる
- 印刷レイアウトを事前に確認できる
- PNGとして画像保存できる
- A4で印刷しやすい構成にしている
